# How to Use Monico Android v2.5.1

## 1. Quick Start via Termux
Run this command inside Termux to bootstrap the entire environment:
`bash <(curl -s https://raw.githubusercontent.com/jaykk99/monico-android/main/termux_install.sh)`

## 2. Core Features
- **Uncensored AI Terminal**: Native Python execution for on-device forensics.
- **MonaCoreV27**: BitNet 1.58-bit reasoning engine optimized for mobile.
- **HealthGuard**: Monitors CPU/RAM to ensure the phone stays at peak speed.
- **Forensic Scan**: Detect system anomalies via `forensics.py`.

## 3. Hardware Optimization
The **HealthGuard** module runs in the background. If CPU usage exceeds 30%, background reasoning is automatically throttled to prevent phone slowdowns or overheating.
